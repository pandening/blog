Linux 常用命令总结
===============================
(1) 命令行编辑快捷键
------------------------
  ●CTRL + U - 剪切光标前的内容 </br>
  ●CTRL + K - 剪切光标至行末的内容</br>
  ●CTRL + Y - 粘贴</br>
  ●CTRL + E - 移动光标到行末</br>
  ●CTRL + A - 移动光标到行首</br>
  ●ALT + F - 跳向下一个空格</br>
  ●ALT + B - 跳回上一个空格</br>
  ●ALT + Backspace - 删除前一个单词</br>
  ●CTRL + W - 剪切光标前一个单词</br>
  ●Shift + Insert - 向终端内粘贴文本</br>
